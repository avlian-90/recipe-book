## Getting Started

# Recipe App - Frontend

## Setup

1. Install dependencies:

```sh
npm install
```

2. Create a .env.local file and add:

NEXT_PUBLIC_API_URL=http://localhost:5000

3. Start the project:

```sh
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
